## 1. What is Queue
- Queues:
    - `Abstract Data Structure`
    - `Higher Level Data Structure`
- Queues means `trail` or `Line`
- Example with `Ticket counter in Lines`. Queues follow to `First-in, First-out (FIFO)` technique.
- Queue Operation:
    - `push()`
    - `pop()`
    - `front()`
    - `size()`
    - `empty()`
    - `back()`
