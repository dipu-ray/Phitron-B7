## 1. Conditional Comparison
```c
if(a > b && b > c) {
    printf("%d", a);
} else if(b > a && c > b) {
    printf("%d", b);
} else {
    printf("%d", c);
}
```
**Given:** `a = 5, b = 1, c = 3`
- `5 > 1 && 1 > 3` ‚Üí `true && false` ‚Üí false
- `1 > 5 && 3 > 1` ‚Üí `false && true` ‚Üí false
- So, `else` block executes ‚Üí Output: `3`

‚úÖ **Correct Answer:** `C) 3`
